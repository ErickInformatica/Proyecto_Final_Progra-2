# Proyecto 2 - Programacion II - UMG
## Sistema de Noticias
Este proyecto esta orientado a poder manegar las noticias del NYT (New York Times) y la capacidad de poder realizar comentarios dentro del sitio web. El sitio, cuenta con un menu facil de navegar y utilizar. 

El proyecto fue realizado con JSP, usando como servidor Jetty y Materialize como framework de dise√±o. 

## Inicio (Noticias)
En esta vista contamos con la visualizacion de las noticias almacenadas en la DB y las cuales pueden ser visualizadas por medio de cards. Si lo deseamos podemos presionar en ver noticias y esto no redigira al detalle de la noticia.

## Detalle Noticia
Al presionar "Ver Noticia", nos redigira de manera automatica al detalle de la noticia donde podremos visualiza la noticias, sus comentarios o bien poder agregar uno.

![enter image description here](https://i.ibb.co/Mcn4p5X/image.png)

## Comentarios y Agregar Comentario
Este apartado lo podemos encontra al ver nuestro Detalle de la Noticia. Aqui podremos visualizar los comentarios creados por los demas usuarios y la capacidad de poder agregar un nuevo comentario.

Para poder agregar un comentario es tan sencillo como agregar nuestro Nombre y agregar el comentario que nosotros deseamos agregar.
